    full_title(page_title) click to toggle source

    Returns the full title on a per-page basis.

    # File app/helpers/application_helper.rb, line 3
def full_title(page_title)
  base_title = "RNRdb"
  if page_title.empty?
    base_title
  else
    "#{base_title} | #{page_title}"
  end
end
